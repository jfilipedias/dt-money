## Getting Started

Para executar esse projeto será necessário ter o [Node.js](https://nodejs.org/) e o [npm](https://www.npmjs.com/) instalados.

Instale as dependências:

```shell
npm i
```

Inicie a aplicação:

```shell
npm run dev
```
